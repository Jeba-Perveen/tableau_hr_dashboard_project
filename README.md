# tableau_hr_dashboard_project

### Overview
This HR Dashboard is a comprehensive human resources analytics platform designed to provide insights into workforce demographics, performance, compensation, and organizational distribution. The dashboard offers both high-level overview metrics and detailed employee-level information.
### Features
##### Overview Dashboard
The main dashboard provides a comprehensive view of the organization's human resources metrics:
### Key Metrics

- Active Employees: Real-time count of current workforce
- Hiring Trends: Historical hiring patterns visualized through trend lines
- Termination Tracking: Monitor employee turnover with visual indicators
- Department Distribution: Breakdown of employees across all organizational departments

### Demographics Analysis

- Gender Distribution: Pie charts showing male/female ratio (89% and 11% split)
- Age Groups: Employee distribution across age brackets (>25, 25-34, 35-44, 55-54, 55+)
- Education Levels: Breakdown by H-sch, Bachelor, Master, and PhD qualifications
- Performance Correlation: Matrix showing relationship between education level and performance ratings (Excellent, Good, Satisfactory, Needs Improvement)

### Compensation Insights

- Education & Gender Income: Salary ranges segmented by education level and gender
- Age & Salary Analysis: Scatter plot visualization showing correlation between age and salary across different roles
- Role-based Compensation: Salary distribution for positions including Finance Manager, IT Manager, Sales Manager, Operations Manager, HR Manager, and HR Assistant

### Geographic Distribution

- Interactive map showing employee distribution across locations
- State-by-state breakdown with employee counts
- HQ vs Branch office visualization

### Details Dashboard
The detailed view provides granular employee information:
#### Employee List Features

- Unique Employee ID: Tracking system for each employee
- Demographics: Name, age, and education level
- Role Information: Job title and department assignment
- Geographic Data: State and city location
- Compensation: Individual salary information
- Employment Status: Hire date and current status
- Tenure: Length of employment visualization

### Filter Capabilities

- Gender-based filtering
- Status filtering (Hired/Terminated)
- Location-based filtering
- Education level filtering
- Hire date filtering

- Workforce Planning: Analyze hiring trends and department distribution for strategic planning
- Diversity Analysis: Track gender and age distribution across the organization

- Performance Management: Correlate education and performance ratings
- Retention Analysis: Monitor termination trends and tenure patterns
- Location Strategy: Optimize resource allocation across geographic locations

### Technical Features

- Interactive filtering system across all dashboard views
- Real-time data visualization with responsive charts
- Drill-down capability from overview to detailed employee records
- Export functionality for reports and further analysis
- Geographic mapping integration
- Trend analysis with historical data visualization

### Navigation

- Overview Tab: High-level metrics and visualizations
- Details Tab: Comprehensive employee listing and filtering
- Info Section: Additional dashboard information
